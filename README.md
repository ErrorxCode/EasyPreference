## Implimentation

In your project level build.gradle
```groovy
  allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
In your module level build.gradle (app)
```groovy
dependencies {
	        implementation 'com.github.ErrorxCode:EasyPreference:1.0.0'
	}
```
